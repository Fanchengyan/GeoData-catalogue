# GeoData-catalogue

常用地理基础数据汇总

## 数据目录

- 植被参数
- 气象数据
- 土壤数据
- 冻土与积雪数据
- 地形与边界数据
  - [DEM数据](data/DEM.md)
  - [北极海岸侵蚀数据](data/北极海岸侵蚀数据.md)
  - [海洋边界数据库](data/海洋边界数据库.md)
  - [冰冻圈范围数据](data/冰冻圈范围.md)
